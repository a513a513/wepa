## 4. SPA vs MPA
<img width="896" height="250" alt="4번이미지" src="https://github.com/user-attachments/assets/baf68c8b-268d-4b1c-82c9-7fb5dcd156be" />
<img width="928" height="247" alt="5번이미지" src="https://github.com/user-attachments/assets/6ba2a0bd-3c8d-40ec-85f6-4a503cebad2a" />

**SPA (Single Page Application)**: 한 개의 페이지로 구성되며, 새로운 페이지 요청이 있을 때 필요 데이터만 받아 갱신합니다[cite: 381, 383].
**MPA (Multi Page Application)**: 여러 개의 페이지로 구성되며, 새로운 페이지를 요청할 때마다 서버에서 렌더링된 전체 정적 리소스를 다운로드합니다[cite: 405, 406].
